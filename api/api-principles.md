# API Principles

## API Principle

The API is not an afterthought. Every schema should be designed as if it may eventually become a public endpoint.

## Governing Constraint

Constitutional Principle #13 (see `governance/principles.md`) and Non-Negotiable #3 (Provenance before publication) both apply directly to the API: no record is exposed publicly through an endpoint unless its publication status, consent, and provenance fields say it may be.
