# Accessibility Hardware

## Required

- tactile controls
- one-handed operation research
- haptic feedback
- audio feedback option
- nonvisual consent state confirmation
- large physical buttons
- status redundancy: light + sound + vibration

## Why This Matters

A blind or low-vision field observer must be able to confirm the device's consent state, that recording started, and that a mark/flag registered — without reading a screen. Status redundancy (light + sound + vibration) is required precisely so no single sense is a single point of failure.
