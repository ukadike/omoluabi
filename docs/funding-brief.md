# Funding Brief

## One-Line Summary

Omoluabi is an open, rules-based editorial intelligence system for trustworthy observation, consent, provenance, accessibility, civic memory, and public knowledge.

## Why Now

AI systems are increasingly able to generate, summarize, classify, and distribute information, but governance, consent, accessibility, provenance, and public accountability often remain afterthoughts. Omoluabi begins from the opposite position: public knowledge systems must be governed before they scale.

## What Fellowship Support Enables

Fellowship support would help transform Omoluabi from a documented research architecture into working MVPs:

1. schema and rule library
2. web engine prototype
3. physical device prototype
4. accessibility-first workflows
5. field observation examples
6. public API roadmap
7. research publication and documentation
