## Why It Matters Beyond Funding

Each fellowship application becomes a forcing function that clarifies scope, research questions, implementation pathways, and public value. The application process itself helps actuate Omoluabi into versions.

## Recognition Strategy

The GitHub repository should make the work legible. Recognition should come from:

- transparent research
- reproducible schemas
- accessible documentation
- clear architecture
- working prototypes
- strong public-interest framing
- disciplined governance
