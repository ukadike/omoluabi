# Omoluabi Vision

Omoluabi is a narrative reasoning platform designed to protect stories from erasure.

It begins with a single question:

> How do we protect stories from erasure?

Before Omoluabi became software, it began as a philosophy about memory, accountability, accessibility, and community stewardship.

This section documents the ideas that guide every design decision within the project.
