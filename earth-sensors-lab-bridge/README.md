# Earth Sensors Lab Bridge

Earth Sensors Lab uses Omoluabi schemas for scientific and environmental observation.

## Bridge Principle

A soil sensor reading is still an observation.

## ESL Uses

- sensor readings
- garden documentation
- student field notes
- accessible STEAM education
- tactile and sonified data
- local environmental archive

## Shared Cards

- Observation Card
- Source Card
- Accessibility Card
- Publication Card
- Memory Card

See `sensor-observation-model.md` for how a sensor reading maps onto these cards, and `examples/esl-sensor-reading.json` for a worked example.
