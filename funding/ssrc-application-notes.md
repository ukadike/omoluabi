# SSRC Application Starter Notes

## Application Frame

Omoluabi is a public-interest research and prototyping project developing trustworthy infrastructure for observation, documentation, accessibility, provenance, consent, and civic knowledge.

## Strong Framing

This is not simply an AI tool. It is a governance-first system for documenting reality under conditions of uncertainty, institutional failure, misrepresentation, accessibility exclusion, and information overload.

## Core Claim

The project asks: What would it mean to build editorial intelligence infrastructure that begins with care, consent, accessibility, and provenance rather than speed, extraction, and scale?

## Deliverables

- Public GitHub repository
- SSL rule library
- stable schema cards
- JSON Schema atlas
- Omoluabi web engine MVP
- Arduino-based physical device MVP plan/prototype
- accessibility workflow documentation
- API roadmap
- research essays / diagrams
- UMADA speculative stress-test sandbox

## Why Kemi

Kemi brings a rare combination of journalism, accessibility, creative technology, systems thinking, public-interest arts leadership, and speculative worldbuilding. Omoluabi grows from decades of editorial, civic, disability, and creative systems experience.
