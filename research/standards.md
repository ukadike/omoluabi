# Standards

Standards Omoluabi designs against, and where each governs:

- **WCAG / WAI-ARIA** — governs `accessibility/` (planned) and the accessibility-metadata requirement in SSL-004.
- **JSON Schema (2020-12)** — governs `schemas/` (planned); every record type is defined as a JSON Schema before any interface is built (Core Architecture Sentence, `docs/vision.md`).
- **OpenAPI** — governs `api/` (planned); every schema is designed as if it may eventually become a public endpoint (API Principle).
- **TinyML / TensorFlow Lite Micro** — governs on-device AI-assist experiments on the Nicla Vision and Nano 33 BLE Sense Rev2 (`device/` planned).
