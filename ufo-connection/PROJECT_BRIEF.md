# Project Brief

## The UFO Connection
### An Omoluabi User Case

This page demonstrates how the Omoluabi framework reads an official public dataset.

The example uses publicly released PURSUE UAP/UFO records from WAR.GOV.

Omoluabi does not determine whether a record represents extraterrestrial technology. It helps readers understand evidence, provenance, uncertainty, accessibility, and public context.

## Ontological Realities

This layer addresses that UAP disclosure is not only a data issue. It also touches reality frameworks: military, scientific, cultural, spiritual, speculative, journalistic, and public-trust realities.

The page should not collapse those into a conclusion. It should separate:

- Evidence
- Interpretation
- Hypothesis
- Belief
- Speculation
