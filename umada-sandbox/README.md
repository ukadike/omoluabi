# UMADA Sandbox

UMADA is not the main Omoluabi project. It is a speculative stress-test environment.

## Purpose

Use UMADA to ask whether Omoluabi can preserve truth under extreme conditions:

- civilization collapse
- lost archives
- extinct languages
- nonhuman testimony
- telepathic communication
- hybrid species
- propaganda
- centuries of historical drift
- alien artifacts
- inaccessible interfaces

## Method

Every UMADA scenario should produce a practical Omoluabi question.

Example:

```text
UMADA question: Can Omoluabi archive telepathic testimony?
Practical question: How does Omoluabi document nonverbal, embodied, or assisted communication today?
Schema implication: Expand Observation Card to include communication mode and translation confidence.
```

See `stress-tests.md` for the stress-test flow and `examples/umada-archive-fragment.json` for a sandbox record.

## Relationship to the UMADA Narrative Project

This sandbox shares its name with the separate UMADA narrative/worldbuilding project (`ukadike/Umada`), where Omoluabi appears as a fictional device used by a future reporter. The two are related in spirit — both use speculative pressure to test what a system can preserve — but this sandbox is an engineering stress-test tool, not canon for that narrative.
