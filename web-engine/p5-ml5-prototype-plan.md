# p5.js / ml5.js Prototype Plan

## Role

p5.js and ml5.js are prototyping tools for the web engine's optional AI-assist experiments, not the production governance system.

## Use For

- sketching interaction ideas
- testing camera/audio input prototypes
- lightweight on-device ML experiments (e.g. labeling suggestions)
- visualizing observation data during design
- rapid UI prototyping before a production framework is chosen
- accessibility prototyping (e.g. testing alt-text generation ideas)
- TinyML bridge experiments shared with `device/` and Earth Sensors Lab work

## Do Not Use For

- governance decisions
- consent decisions
- risk decisions
- accessibility sign-off
- publication status decisions

Any AI assistance surfaced through a p5.js/ml5.js prototype is advisory only, per `governance/ai-permissions.md`. A human reviewer makes the decision.
